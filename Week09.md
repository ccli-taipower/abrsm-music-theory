## 📖 概念講解

### 節奏創作（Composing a Rhythm）

Grade 1 考試有一道題是給你前兩小節的節奏，要你**創作後兩小節作為「回應」（answering phrase）**。這道題看起來難，但只要掌握幾個規則，其實很有把握。

---

### 節奏創作的連桿規則

**規則一：以一拍為單位連桿**
八分音符（quaver）和十六分音符（semiquaver）要以**一個四分音符的時值**為一組來連桿：
- 兩個八分音符 = 一拍 → 連成一組 ✓
- 四個十六分音符 = 一拍 → 連成一組 ✓
- 兩個十六分音符 + 一個八分音符 = 一拍 → 連成一組 ✓

**規則二：4/4 拍不可跨越中線**
在 4/4 拍中，連桿不可以從第 2 拍跨越到第 3 拍（即不可以跨越小節的正中間）。前兩拍一組，後兩拍一組。

**規則三：盡量避免連結線（tie）**
如果能用一個完整的音符表達，就不要用連結線去「拼湊」出同樣的時值，除非確實必要。

### 答句樂句的原則

創作答句時，要讓它和「問句」有**音樂上的關聯**，但又不是完全的複製：

1. **保持相同的「情緒」**：問句如果是活潑的八分音符節奏，答句就不要突然變成緩慢的全音符
2. **開頭可以呼應問句**：答句可以用和問句相同或相似的節奏開頭幾拍，讓兩者產生連結感
3. **結尾要有完結感**：答句最後一個音應該是**四分音符或更長的音符**，不能用八分音符或十六分音符結尾，否則聽起來像「沒說完就停了」
4. **不要完全複製問句**：完全一樣的節奏算是缺乏創意，也不符合「問答」的精神

### Grade 1 必考義大利文術語

Grade 1 考試有一整題要求辨認演奏術語的意義。以下是完整列表，按類別整理：

---

#### 速度術語（Tempo）

| 義大利文 | 中文意思 | 補充說明 |
|---|---|---|
| Adagio | 緩慢地 | 非常慢的速度 |
| Lento | 緩慢地 | 和 Adagio 相近，都是慢速 |
| Andante | 行板，以步行的速度 | 中等偏慢的速度，像悠閒地走路 |
| Allegretto | 稍快地 | 比 Allegro 慢一些 |
| Allegro | 快速地、愉快地 | 原意是「快樂的」，常見的快速標記 |
| Accelerando (accel.) | 漸漸加快 | 速度逐漸變快，直到另有指示 |
| Rallentando (rall.) | 漸漸減慢 | 速度逐漸變慢 |
| Ritardando (ritard. / rit.) | 漸漸減慢 | 和 Rallentando 意思非常相近 |
| Ritenuto (riten. / rit.) | 立即放慢 | 與 Rallentando 不同，是馬上慢下來，不是漸漸慢 |

**記憶技巧：**
- 慢速組：Adagio、Lento（兩個都是「慢」，可以一起記）
- 快速組：Allegro（快）、Allegretto（稍快，少了個字母，速度也少一點）
- 變慢組：Rallentando、Ritardando（漸漸慢）vs Ritenuto（立即慢）

---

#### 力度術語（Dynamics）

| 符號 / 義大利文 | 中文意思 | 補充說明 |
|---|---|---|
| **p** (piano) | 輕聲地 | p 是義大利文 piano（輕）的縮寫 |
| **pp** (pianissimo) | 非常輕聲地 | 比 p 更輕，雙 p |
| **mp** (mezzo piano) | 稍輕聲地 | mezzo = 「中等」、「一半」 |
| **mf** (mezzo forte) | 稍響地 | 中等力度，最常見的力度標記之一 |
| **f** (forte) | 響亮地 | f 是 forte（強）的縮寫 |
| **ff** (fortissimo) | 非常響亮地 | 比 f 更強，雙 f |
| Crescendo (cresc.) | 漸漸變響 | 也可以用「<」符號表示 |
| Decrescendo (decresc.) | 漸漸變輕 | 也可以用「>」符號，或 Diminuendo（dim.）|

**從輕到響的順序：**
pp → p → mp → mf → f → ff

**記憶技巧：** piano 這個樂器的名字就來自「強弱（piano e forte）」，因為它可以彈出輕重不同的音量！

---

#### 表情與其他術語

| 義大利文（或縮寫） | 中文意思 | 補充說明 |
|---|---|---|
| Cantabile | 如歌般地 | 要演奏得像在唱歌一樣流暢優美 |
| Fine | 終止（結束） | 標記在曲子真正結束的地方 |
| Da Capo (D.C.) | 從頭重複 | 從曲子最開頭再演奏一遍 |
| Dal Segno (D.S.) | 從記號處重複 | 從 **%**（Segno 記號）處開始重複 |
| poco | 少許、稍微 | 修飾其他術語，如 poco allegro = 稍微快一點 |
| mezzo | 中等、一半 | 用於 mezzo forte、mezzo piano |
| 8va | 高八度演奏 | 標記在音符上方，表示要比寫的音高一個八度演奏 |
| Fermata (𝄐 / U) | 延長記號 | 音符或休止符上的小帽子，表示要比正常時值延長 |

### 最後的考試準備建議

1. **每個調的音階都要能從頭到尾寫出來**：C、G、D、F 大調，上行和下行各一遍
2. **主三和弦的四個調都要背熟**：C(C-E-G)、G(G-B-D)、D(D-F♯-A)、F(F-A-C)
3. **術語要熟到看到馬上能說出意思**：可以把術語卡片（flashcard）寫在小紙片上，每天翻看
4. **節奏創作要動手練習**：光看不夠，要自己用筆在紙上寫節奏，並數拍子確認正確
5. **連桿規則要能自動執行**：看到八分音符就習慣性地以一拍為單位分組

## ✏️ 本週練習題

1. 請解釋 Rallentando 和 Ritenuto 有什麼不同？

2. 力度記號從最弱排到最強的正確順序是什麼？（從 pp 到 ff）

3. 「D.C.（Da Capo）」告訴演奏者要做什麼？

4. 在節奏創作中，為什麼答句的最後一個音不能是八分音符？

5. 請把以下術語配對到正確的意思：
   - (a) Cantabile
   - (b) Andante
   - (c) poco
   - (d) Fermata (U)

   選項：① 以步行的速度 ② 稍微 ③ 延長記號 ④ 如歌般地

## ✅ 解答
1. 兩者都是「變慢」，但方式不同：**Rallentando（漸漸減慢）** 是**逐漸**慢下來，就像車子輕輕踩煞車；**Ritenuto（立即放慢）** 是**馬上**就慢下來，就像突然踩緊煞車。考試如果問兩者的差異，這個「漸漸 vs 立即」是關鍵。

2. 從最弱到最強：**pp → p → mp → mf → f → ff**

3. D.C.（Da Capo）告訴演奏者要**從曲子最開頭重新演奏**，一直演奏到標有「Fine（終止）」的地方或到曲子結束。

4. 因為答句最後用八分音符結尾，聽起來像「還沒說完就停下來」，沒有完結感。答句最後應該用**四分音符或更長的音符**（如二分音符），讓節奏有一個清楚的結束感，就像一個句子要以句號結束，不能以逗號結束。

5. 配對答案：
   - (a) Cantabile → **④ 如歌般地**
   - (b) Andante → **① 以步行的速度**
   - (c) poco → **② 稍微**
   - (d) Fermata → **③ 延長記號**

## 💡 考試技巧

- **術語題是「送分題」**：只要考前把所有術語背熟，這道題幾乎可以拿滿分。建議用閃卡（flashcard）每天各背 5 個，9 週下來全部覆蓋到。
- **Rallentando vs Ritardando**：兩者意思非常接近（都是「漸漸慢」），考試通常不要求分辨兩者差異，只要知道都是「漸慢」就可以。Ritenuto 是「立即慢」，這個才要特別區分。
- **節奏創作滿分技巧**：
  1. 先確認拍號，搞清楚每小節幾拍
  2. 每寫完一小節就數一遍拍數，確認正確
  3. 最後一個音用四分音符或更長的音符
  4. 連桿要正確（一拍一組）
- **考試時間管理**：Grade 1 試卷通常有 1.5 小時。音階和主三和弦部分要快，節奏創作和術語部分要細心。
