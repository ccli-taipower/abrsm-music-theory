## 📖 概念講解

### 1. Grade 3 音程複習

Grade 3 中你學過：
- **度數（number）**：兩音之間相差幾個音級（含兩端）
- **質性（quality）**：Major（大）、Minor（小）、Perfect（完全）

| 度數 | 可能的質性 |
|------|----------|
| 2、3、6、7 度 | Major（大）或 Minor（小） |
| 4、5、8 度 | Perfect（完全） |
| 1 度（同音） | Perfect |

### 2. Grade 4 新增：增音程與減音程

Grade 4 在上述基礎上加入兩種新的質性：

#### 非完全音程（2、3、6、7 度）的變化規則：

```
Diminished（減）← Minor - 半音 │ Minor + 半音 → Major → Major + 半音 → Augmented（增）
```

| 操作 | 結果 |
|------|------|
| Major（大）+ 再加半音 | **Augmented（增）** |
| Major（大）- 減半音 | Minor（小） |
| Minor（小）+ 加半音 | Major（大） |
| Minor（小）- 再減半音 | **Diminished（減）** |

#### 完全音程（4、5、8 度）的變化規則：

```
Diminished（減）← Perfect - 半音 │ Perfect + 半音 → Augmented（增）
```

| 操作 | 結果 |
|------|------|
| Perfect（完全）+ 加半音 | **Augmented（增）** |
| Perfect（完全）- 減半音 | **Diminished（減）** |

> 重要：完全音程**沒有** Major 或 Minor 之分，只有 Perfect、Augmented、Diminished。

### 3. 臨時記號對音程大小的影響

這是 Grade 4 的核心難點，原文特別強調：

> 「Be aware that **a flat on the lower note makes the interval bigger** and **a sharp on the lower note makes the interval smaller**. It is helpful to visualise this on a piano keyboard.」

**為什麼？**
想像兩個音之間的距離：
- 下方音有**降號（♭）**：下方音「往下移」了，兩音之間的距離**變大**
- 下方音有**升號（♯）**：下方音「往上移」了，兩音之間的距離**變小**

反之：
- 上方音有降號（♭）：音程**變小**
- 上方音有升號（♯）：音程**變大**

**鋼琴鍵盤視覺化：**
想像 C 到 E：
- C 到 E = 大三度（Major 3rd）
- C♯ 到 E = 小三度（Minor 3rd）— 下方音升高，距離縮小
- C♭ 到 E = 增三度（Augmented 3rd）— 下方音降低，距離擴大

### 4. 四步驟判斷法

根據原文的步驟（稍作整理）：

**步驟一：** 把下方音當作**調性的主音（tonic）**，確認其大調調號。

**步驟二：** 觀察調號中的升降號，看它們如何影響上方那個音（**暫時忽略臨時記號**）。

**步驟三：** 判斷上方音在該大調音階中是否存在——如果存在，目前是 **Major 或 Perfect**；如果不在，需要判斷差了多少。

**步驟四：** 考慮**臨時記號**的影響——臨時記號是讓音程變大還是變小，再對應到表格得出最終答案。

### 5. 實際例子

#### 例 1：C 到 E♭（三度）
1. 以 C 為主音，C 大調無升降號
2. 忽略臨時記號：C 到 E = 大三度（Major 3rd）
3. 上方音 E 加了 ♭，音程**縮小**半音
4. Major - 半音 = **Minor 3rd（小三度）** → 答案：**小三度（Minor 3rd）**

#### 例 2：C♭ 到 E（三度）
1. 以 C♭ 為主音
2. 忽略臨時記號（把 C♭ 先當 C 處理）：C 到 E = 大三度
3. 下方音 C 有 ♭，音程**變大**半音
4. Major + 半音 = **Augmented 3rd（增三度）** → 答案：**增三度（Augmented 3rd）**

#### 例 3：C 到 G♭（五度）
1. 以 C 為主音，C 大調無升降號
2. 忽略臨時記號：C 到 G = 完全五度（Perfect 5th）
3. 上方音 G 有 ♭，音程**縮小**半音
4. Perfect - 半音 = **Diminished 5th（減五度）** → 答案：**減五度（Diminished 5th）**

#### 例 4：F♯ 到 B（四度）
1. 以 F♯ 為主音（視為 F 大調，但下方音是 F♯）
2. F 大調中，B 是 B♭；但實際音符是 B♮（無降號）
   所以 F 到 B♭ = Perfect 4th；F 到 B♮ 比完全四度**大**了半音
3. 現在 F♯ 到 B：下方音 F 有 ♯，音程**縮小**半音
4. 前面已是比 Perfect 4th 大半音（= Augmented 4th），再縮小半音 = 回到 **Perfect 4th（完全四度）** → 答案：**完全四度（Perfect 4th）**

## ✏️ 本週練習題

**題 1：** 完全音程（Perfect interval）加半音變成什麼？減半音又變成什麼？

**題 2：** 大音程（Major interval）減半音變成什麼？再繼續減半音呢？

**題 3：** 下方音有升號（♯），對音程大小有什麼影響？為什麼？

**題 4：** 判斷以下音程的質性（請用四步驟法）：
- D 到 F♯（三度）

**題 5：** 判斷以下音程的質性：
- A♭ 到 E♭（五度）
（提示：先考慮 A 到 E，再考慮兩個臨時記號的效果）

---

## ✅ 解答

**題 1：**
- Perfect + 半音 = **Augmented（增）**
- Perfect - 半音 = **Diminished（減）**

**題 2：**
- Major - 半音 = **Minor（小）**
- Minor - 半音 = **Diminished（減）**

**題 3：** 下方音有升號，代表下方音**往上移動**了半音，兩音之間的距離**縮小**半音，音程**變小**。可以用鋼琴鍵盤想像：原來的距離縮短了。

**題 4：** D 到 F♯（三度）
1. 以 D 為主音，D 大調有 F♯ 和 C♯
2. D 大調音階中，第 3 音就是 F♯
3. 所以 D 到 F♯ = **大三度（Major 3rd）**（直接在 D 大調音階中找到）
答案：**Major 3rd（大三度）**

**題 5：** A♭ 到 E♭（五度）
1. 以 A♭ 為主音（把 A♭ 先當 A 處理）
2. 忽略臨時記號：A 到 E = 完全五度（Perfect 5th）
3. 下方音有 ♭（A♭）→ 音程**變大**半音 → 比 Perfect 5th 大半音 = Augmented 5th
4. 上方音有 ♭（E♭）→ 音程**縮小**半音 → Augmented - 半音 = 回到 **Perfect 5th**
答案：**Perfect 5th（完全五度）**

## 💡 考試技巧

1. **先判斷度數（幾度），再判斷質性**——度數從下方音數到上方音（含兩端），不要跳過。
2. 記住「下方音 ♭ → 音程變大；下方音 ♯ → 音程變小」——這個口訣在考試中非常實用。
3. 遇到複雜的臨時記號，先**忽略所有臨時記號**，判斷基本音程；再**逐一加入**臨時記號調整。
4. 完全音程（4、5、8 度）**沒有 Major 或 Minor**，只有 Perfect、Augmented、Diminished——不要亂用。
