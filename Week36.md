## 📖 Grade 4 核心複習

### 中音譜號（Alto Clef）

- 外形：C 譜號
- **中央 C 在第三線**
- 主要用於：中音提琴（Viola）

各線間的音名（從低到高）：
```
第五線 = F
第四間 = E
第四線 = D
第三間 = C（中央 C 以上的 C）
第三線 = B（中央 C 以下的 B）
  → 第三線 = Middle C（中央 C）
第二間 = A
第二線 = G
第一間 = F
第一線 = E
```

（複習：次中音譜號 Tenor clef 中央 C 在**第四線**，是 Grade 5 新內容）

### 增減音程（Augmented & Diminished Intervals）

| 品質 | 比較基準 | 半音差 |
|------|---------|-------|
| Augmented（增） | 比 Major 或 Perfect 多 1 個半音 | +1 |
| Diminished（減） | 比 Minor 或 Perfect 少 1 個半音 | -1 |

例：
- 完全五度（Perfect 5th）= C 到 G（7 個半音）
- 增五度（Augmented 5th）= C 到 G#（8 個半音）
- 減五度（Diminished 5th）= C 到 Gb（6 個半音）

### Grade 4 裝飾音符號

| 符號 | 名稱 | 演奏方式 |
|------|------|---------|
| tr | Trill（顫音） | 與上方鄰音快速交替 |
| ∿（或 ） | Mordent（震音） | 主音 → 上鄰音 → 主音（或向下） |
| ∞ 型（S 形）| Turn（回音） | 上鄰音 → 主音 → 下鄰音 → 主音 |
| 小八分音符（有斜線）| Acciaccatura（短倚音）| 快速滑過 |

### I、IV、V 和弦（Grade 4 基礎）

Grade 4 確立了以羅馬數字命名和弦的系統，Grade 5 在此基礎上加入轉位（a/b/c）。

### Grade 4 自測（共 3 題）

**Q1：** 中音譜號（Alto clef）的中央 C 在第幾線？

**Q2：** C 到 F#（升 F）是什麼音程？（提示：C 到 F 是完全四度，再升半音）

**Q3：** Acciaccatura（短倚音）與 Appoggiatura（長倚音）的主要差別是什麼？

**解答：**
- A1：**第三線**
- A2：**增四度（Augmented 4th）**（完全四度 + 半音）
- A3：短倚音（Acciaccatura）的符號是**帶斜線**的小音符，演奏非常短暫、幾乎不佔時值；長倚音（Appoggiatura）**不帶斜線**，佔用主音的一半時值。

## 📖 Grade 5 核心複習

### 次中音譜號（Tenor Clef）

- 中央 C 在**第四線**（比 Alto clef 高一條線）
- 主要用於：大提琴高音區、長號

### 和弦轉位（a/b/c）

| 符號 | 名稱 | 最低音 |
|------|------|-------|
| a | 根音位置 | 根音（Root） |
| b | 第一轉位 | 三音（Third） |
| c | 第二轉位 | 五音（Fifth） |

### 終止式（三種）

| 終止式 | 進行 | 感覺 |
|--------|------|------|
| 完全終止（Perfect） | V → I | 完結 |
| 變格終止（Plagal） | IV → I | 柔和（Amen） |
| 不完全終止（Imperfect） | ? → V | 懸掛 |

### 8 小節旋律結構

| 小節 | 功能 | 和弦 |
|------|------|------|
| 1 | 確立調性 | I |
| 2–3 | 發展動機 | I / II / IV |
| 4 | 問句結尾 | V（不完全終止） |
| 5 | 重確立調性 | I |
| 6–7 | 收束準備 | II / IV |
| 8 | 答句結尾 | V → I（完全終止） |

### Grade 5 自測（共 5 題）

**Q4：** Gb 大調的關係小調是什麼？

**Q5：** C 大調 IVb 的最低音是什麼？

**Q6：** 什麼是「Compound Perfect 5th」？等於幾度？

**Q7：** 短縮總譜（short score）中，低音譜號裡符桿朝上的聲部是哪個？

**Q8：** 8 小節旋律的第 8 小節應以哪種終止式結束？

**解答：**
- A4：**Eb 小調**
- A5：IVb = Chord IV 第一轉位，Chord IV = F-A-C，三音是 A，所以最低音是 **A**
- A6：Compound Perfect 5th = 完全五度超過一個八度，即 **Perfect 12th（完全十二度）**
- A7：**Tenor（男高音）**（Tenor 在低音譜號的上方，符桿朝上）
- A8：**完全終止（Perfect cadence）= V → I**

## 📖 模擬考試指導

### 考試形式說明

ABRSM Grade 5 樂理考試目前以**線上考試（online exam）** 為主，採**選擇題**形式。

- 考試時間：**2 小時**（Grade 5）
- 題目語言：可選繁體中文（在報名時選擇語言選項）

### 建議作答順序

1. **先快速瀏覽全卷**（約 5 分鐘）：了解題目類型，標記自己最熟悉的題目。
2. **先答有把握的題目**：不要在不確定的題目上卡太久，先拿確定的分數。
3. **標記並回頭解決難題**：線上考試通常允許標記題目，先跳過，最後再回頭。
4. **最後 10 分鐘全面檢查**：

### 快速檢查清單（考試結束前確認）

- [ ] 所有調號題：升降記號的**順序**和**位置**正確
- [ ] 音程題：記得**兩端都算**在內
- [ ] 旋律小音階：下行有沒有把升號**還原**
- [ ] 次中音譜號：中央 C 在**第四線**（不是第三線）
- [ ] 和弦轉位符號：用**小寫** a、b、c（不要用大寫）
- [ ] 不規則分拍：確認是「5/6/7 對應 4」還是「9 對應 8」
- [ ] 終止式：完全終止是 **V→I**，變格是 **IV→I**，不完全是 **?→V**
- [ ] 所有題目都有作答（沒有遺漏）

## 📋 Grade 5 關鍵知識點最終速覽

| 主題 | 核心記憶點 |
|------|---------|
| 次中音譜號 | 中央 C 在**第四線** |
| 6 升大調 | F# 大調（關係小調：D# 小調） |
| 6 降大調 | Gb 大調（關係小調：Eb 小調） |
| 小調第 7 音 | 降號→還原記號；升號→重升號 |
| 不規則拍號 | 5/8 = 2+3 或 3+2；7/4 = 3+4 或 4+3 |
| 不規則分拍 | 5,6,7 → 佔 4 個音的時值；9 → 佔 8 個音的時值 |
| 短縮總譜 | S/A 在高音譜號；T/B 在低音譜號；符桿上=上聲部，下=下聲部 |
| 開放總譜 | Tenor 用帶「8」的高音譜號，實際低一個八度 |
| 複音程 | Compound + 原音程名（如 Compound Perfect 4th） |
| 和弦轉位 | a=根音在下，b=三音在下，c=五音在下 |
| 完全終止 | V → I |
| 變格終止 | IV → I（Amen） |
| 不完全終止 | ? → V（懸掛） |
| 旋律創作 | 第 4 小節 Chord V；第 8 小節 V→I；結尾音符不短於四分音符 |
| Trill 時代差異 | 古典（海頓/莫札特以前）→ 從上方鄰音；現代 → 從旋律音 |
