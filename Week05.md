# 第 05 週：臨時記號、全音與半音
**對應級別：Grade 1**

## 🎯 本週學習目標
完成本週後，你將能夠：

## 📖 概念講解

### 升記號（Sharp ♯）

**升記號（sharp）**，符號是 **♯**，寫在音符的左邊。它的作用是：**把這個音的音高升高一個半音（semitone）**。

想像鋼琴鍵盤，C 的右邊緊鄰一個黑鍵，那個黑鍵就叫做 **C♯（C sharp）**。升記號就是讓你「往右移一個鍵」（不論是白鍵還是黑鍵，只要是最近的那個鍵就是升一個半音）。

---

### 降記號（Flat ♭）

**降記號（flat）**，符號是 **♭**，也寫在音符的左邊。它的作用是：**把這個音的音高降低一個半音**。

例如，B 的左邊緊鄰一個黑鍵，那個黑鍵叫做 **B♭（B flat）**。降記號就是讓你「往左移一個鍵」。

---

### 還原記號（Natural ♮）

**還原記號（natural）**，符號是 **♮**，寫在音符的左邊。它的作用是：**取消之前的升記號或降記號，讓音符回到原來的音高**。

例如，一小節裡先出現了 F♯，後來出現了還原記號 ♮F，就表示這個 F 要回到沒有升降的原始 F。

### 臨時記號的有效範圍

這是 Grade 1 必考的重要規則，請務必記牢：

> **臨時記號只在它出現的那一個小節內有效，而且只影響同一音高（同一個八度）的後續音符。小節線一過，臨時記號自動失效，音符恢復到調號所規定的音高。**

舉例說明：
- 小節 1：出現了 F♯，則該小節後面所有的 F（同音高）都自動是 F♯，不需要重複寫 ♯。
- 小節 2 開始：F 恢復正常（回到調號的設定），需要重新標注才能是 F♯。
- 另外：臨時記號**不影響其他八度**的同名音符。

---

### 半音（Semitone）與全音（Tone）

這兩個概念最好用**鋼琴鍵盤**來理解。

**半音（semitone）= 最近的相鄰鍵**

鋼琴上任意一個鍵，往右或往左移動**一個鍵**（不管是白鍵還是黑鍵），就移動了一個半音。半音是西方音樂體系中最小的音程單位。

例如：
- C → C♯（或 D♭）：一個半音
- E → F：一個半音（中間沒有黑鍵！）
- B → C：一個半音（中間沒有黑鍵！）

**全音（tone）= 兩個半音**

從一個鍵移動**兩個鍵**就是一個全音。

例如：
- C → D：一個全音（C→C♯→D，跳過了中間的黑鍵）
- D → E：一個全音
- C♯（D♭）→ D♯（E♭）：一個全音

[NOTATION:w05_semitone_tone]

### 特別重要：沒有黑鍵的地方

在鋼琴鍵盤上，大多數白鍵之間都有黑鍵，但有兩個例外：

1. **E 和 F 之間**：沒有黑鍵，所以 E 到 F 直接就是**半音**
2. **B 和 C 之間**：沒有黑鍵，所以 B 到 C 直接就是**半音**

這兩個地方非常重要，在大音階的學習（下週）和考試中都會反覆用到。記憶法：**「EF、BC 不分家」**（它們之間只有半音距離）。

---

### 異名同音（Enharmonic）

這是一個有趣的概念：**同一個音可以有兩個不同的名字**。

例如，鋼琴上 F♯ 和 G♭ 所代表的是完全相同的那個黑鍵——按下去的音完全一樣，但在不同的音樂脈絡中會用不同的名字來稱呼它。這種現象叫做**異名同音（enharmonic equivalents）**。

常見的異名同音配對：
| 升記號名 | 降記號名 | 說明 |
|---|---|---|
| C♯ | D♭ | 同一個黑鍵 |
| D♯ | E♭ | 同一個黑鍵 |
| F♯ | G♭ | 同一個黑鍵 |
| G♯ | A♭ | 同一個黑鍵 |
| A♯ | B♭ | 同一個黑鍵 |

還有兩個白鍵也有異名：
- **E♯ = F**（E 升高一個半音就是 F，因為中間沒有黑鍵）
- **B♯ = C**（同理）
- **C♭ = B**（C 降低一個半音就是 B）
- **F♭ = E**（同理）

## ✏️ 本週練習題

1. 請問「臨時記號只在本小節有效」是什麼意思？如果第一小節出現了 B♭，在第二小節的開始，那個 B 還是 B♭ 嗎？

2. 在鋼琴鍵盤上，E 和 F 之間有黑鍵嗎？從 E 到 F 是幾個半音？是半音還是全音？

3. F♯ 的異名同音（enharmonic equivalent）是什麼？

4. 一個全音等於幾個半音？從 C 到 D 是全音還是半音？請說明理由。

5. 在同一個小節裡，一個 G♯ 出現後，若之後再次出現 G，請問這個 G 是：
   - (A) 自動仍是 G♯，不需要另外標記
   - (B) 回到了正常的 G（沒有升降），需要標注還原記號才算是普通 G
   - (C) 自動變成 G♭

## ✅ 解答
1. 意思是：這個臨時記號的效力在小節線之後就**自動消失**。在第二小節開頭，那個 B **不再是 B♭**，而是回到調號規定的音高（在 C 大調中就是普通的 B）。如果在第二小節還要用 B♭，必須重新標注 ♭ 記號。

2. E 和 F 之間**沒有黑鍵**。從 E 到 F 是 **1 個半音**（semitone），是半音，不是全音。

3. F♯ 的異名同音是 **G♭**。這兩個音在鋼琴上按的是完全相同的黑鍵。

4. 一個全音等於 **2 個半音**。從 C 到 D 是**全音**，因為中間有一個黑鍵（C♯/D♭），所以 C→C♯ 是一個半音，C♯→D 又是一個半音，共兩個半音。

5. **(A) 正確**。在同一個小節內，臨時記號一旦出現，對該小節後續**相同音高**的音符都持續有效，不需要重複標記。所以那個後出現的 G，在臨時記號沒有被還原記號取消的情況下，仍然是 G♯。

---

## 💡 考試技巧

- **臨時記號有效範圍**是非常高頻考點。記住兩個關鍵：(1) 只在本小節有效，(2) 只影響同音高，不影響其他八度。
- **E-F 和 B-C 是半音**：大音階題型中會一再用到這個知識，現在記住可以省下很多時間。
- **異名同音考題**：通常直接問「F♯ 的異名同音是什麼」，把五對異名同音（C♯/D♭、D♯/E♭、F♯/G♭、G♯/A♭、A♯/B♭）都記熟。
- **還原記號的位置**：寫在音符左邊，緊貼著音符，不是寫在旁邊很遠的地方。
