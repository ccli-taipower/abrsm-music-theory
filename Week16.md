# 第 16 週：複拍子音符分組 + 單複拍子互換
**對應級別：Grade 3**
**本週預計學習時間：每天約 20 分鐘 × 5 天**

---

## 🎯 本週學習目標
完成本週後，你將能夠：
- 在複拍子中正確分組音符，讓每個附點四分音符拍「自成一組」
- 正確書寫複拍子中的休止符（使用附點休止符）
- 理解複拍子中連結線（tie）的使用時機
- 把複拍子旋律轉換為單拍子（加三連音記號、去附點）
- 把單拍子旋律轉換為複拍子（去三連音記號、加附點）

---

## 📖 概念講解

### 1. 複拍子的音符分組原則

複拍子中的分組原則與單拍子相同，只是「拍」的單位換成了附點四分音符：

**核心原則：每一個大拍（附點四分音符 = 3 個八分音符）必須在自己的範圍內完成，不可以讓連桿或音群跨越到下一個大拍。**

**6/8 中的正確分組範例：**

若有六個八分音符（一整小節），正確做法是：
- 前 3 個八分音符連成一組（第 1 大拍）
- 後 3 個八分音符連成一組（第 2 大拍）

若有混合音符（如一個四分音符接一個八分音符），也要確認它們加起來等於一個附點四分音符（3 個八分音符的時值），才能放在同一大拍內。

### 2. 複拍子中使用連結線（Tie）

有時候，一個音符的時值跨越兩個大拍，這時候**不能畫單一音符跨越大拍**，必須使用**連結線（tie）**把兩個音符連起來。

例如在 6/8 中，如果要表達一個持續 4 個八分音符的音，寫法是：
- 附點四分音符（3 個八分音符，第 1 大拍）**連結（tie）** 八分音符（第 2 大拍的第 1 個八分音符）
- 這樣清楚顯示大拍的邊界在哪裡

**連結線（tie）vs. 圓滑線（slur）的區別：**
- 連結線：兩個**相同音高**的音符，連在一起計時值（持音）
- 圓滑線：兩個**不同音高**的音符，連在一起演奏（圓滑）

### 3. 複拍子中的休止符

複拍子中的休止符有特別規定：

**每一個完整大拍的靜默，要用附點音符對應的休止符。**

具體規則：
- 一個完整大拍靜默（= 附點四分音符的時值）→ 使用**附點四分休止符（dotted crotchet rest）**
- 兩個大拍靜默（= 附點二分音符的時值）→ 使用**附點二分休止符**
- 整小節靜默 → 使用**全休止符**（全休止符代表「整個小節靜默」，不論拍號）

在 6/8 中，若第 1 大拍靜默，第 2 大拍有音符，不可以用一個「附點二分休止符」把整小節的前半都蓋掉，而要用一個附點四分休止符表示第 1 大拍的靜默。

### 4. 複拍子 → 單拍子的轉換

**轉換規則：**
複拍子中，一大拍 = 3 個八分音符（自然成組，無三連音記號）。
轉換成單拍子後，3 個八分音符就變成三連音，**必須加上三連音括號和數字 3**，同時**去掉附點**。

**步驟：**
1. 找到複拍子中每一個大拍（每組 3 個八分音符）
2. 把每組 3 個八分音符加上三連音括號（number 3）
3. 把拍號改為對應的單拍子（6/8 → 2/4，9/8 → 3/4，12/8 → 4/4）
4. 若有附點音符，去掉附點（例：附點四分音符 → 四分音符）

**舉例（6/8 → 2/4）：**
```
原始（6/8）：[♩ ♩ ♩ | ♩. ♩]
轉換（2/4）：[³♩ ♩ ♩ | ♩ ♩²(tie)]
```

### 5. 單拍子 → 複拍子的轉換

**轉換規則：**
單拍子中，三個音符的三連音 → 複拍子中，這三個音符自然地成組（無需三連音記號），同時**加附點**。

**步驟：**
1. 找到單拍子中每一組三連音
2. 去掉三連音括號和數字 3
3. 把拍號改為對應的複拍子（2/4 → 6/8，3/4 → 9/8，4/4 → 12/8）
4. 若有非三連音的附點音符組合，加上附點

**實際例子：**
```
原始（2/4）：[³♩ ♩ ♩ | ³♩ ♩ ♩]（兩組三連音）
轉換（6/8）：[♩ ♩ ♩ | ♩ ♩ ♩]（六個八分音符，自然分成兩大拍，無三連音記號）
```

### 6. 記住對應關係

| 單拍子 | 對應複拍子 |
|-------|----------|
| 2/4 | 6/8 |
| 3/4 | 9/8 |
| 4/4 | 12/8 |
| 2/2 | 6/4（較少見，但存在） |

## ✏️ 本週練習題

**題目一：**
在 6/8 拍中，為什麼「一個四分音符接一個八分音符」可以在同一個大拍內？這樣的組合加起來是幾個八分音符的時值？

**題目二：**
在 6/8 拍中，如果第一大拍整個靜默，應該用什麼休止符來表示？

**題目三：**
把以下 6/8 旋律描述轉換為 2/4（只需描述轉換後的音符樣式）：
「6/8 中，第一大拍是三個八分音符（do re mi），第二大拍是一個附點四分音符（fa）接一個八分音符（sol）。」

**題目四：**
把以下 2/4 旋律轉換為 6/8：
「2/4 中，兩小節，每小節都是一組四分音符三連音（3 個四分音符三連音）。」

**題目五：**
連結線（tie）和圓滑線（slur）外觀相似，請說出分辨它們的方法。

---

## ✅ 解答

**題一：** 四分音符 = 2 個八分音符，八分音符 = 1 個八分音符，合計 **3 個八分音符**的時值，恰好等於一個大拍（附點四分音符），所以可以在同一大拍內。

**題二：** 使用一個**附點四分休止符（dotted crotchet rest）**，代表 3 個八分音符的靜默，即一整個大拍。

**題三：** 轉換為 2/4 後：
- 第一大拍的三個八分音符（do re mi）→ 加上三連音記號，變成**三連音三個八分音符（do re mi）**
- 第二大拍的附點四分音符（fa）→ 去掉附點，變成**四分音符**；八分音符（sol）→ **八分音符**（但現在這個八分音符是獨立的，非三連音）
- 實際上第二大拍的「附點四分 + 八分」在 2/4 中寫成「四分音符（fa）+ 八分音符（sol）」不需三連音，因為這是自然的節奏填充。

**題四：** 轉換為 6/8 後：
原本每小節是「一組四分音符三連音（3 個四分音符）」，去掉三連音記號，4/4 中的三連音是以二分音符為基準。但題目說的是 2/4 中每小節一組四分音符三連音……在 2/4 中，一組四分音符三連音佔 2 個四分音符（即一整小節的時值），轉換為 6/8 後，每小節有 2 大拍，各含 3 個八分音符。原三連音的 3 個四分音符，每個相當於 2 個八分音符，但整組等於一小節……此處依照標準規則：**去掉三連音括號、把三個四分音符改為三個帶附點感的等份。** 在 6/8 中，每小節可以寫成：附點四分音符 + 附點四分音符（2 大拍），或視具體情況分成 6 個八分音符。

**題五：** 分辨方法：連結線連接的是**兩個相同音高**的音符（例如兩個 C），目的是延長時值；圓滑線連接的是**不同音高**的音符（例如 C 到 E），目的是圓滑演奏。外觀上都是弧形線，但看音符音高是否相同就可以分辨。

## 💡 考試技巧

1. **轉換題要改拍號：** 把 6/8 轉換為 2/4 時，別忘了把拍號也一起改，只改音符不改拍號會失分。

2. **複拍子中的全休止符：** 無論什麼拍子，整小節靜默永遠用**全休止符**，這個規則在複拍子也適用。

3. **附點音符的轉換：** 複拍子中的附點四分音符，轉換到單拍子就是三連音（去掉附點）；反之單拍子三連音轉到複拍子就加附點。這是考試中「互換」題的核心變換。
