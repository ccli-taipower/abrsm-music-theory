# 第 27 週：次中音譜號（Tenor Clef）+ 六個升降號大小調

**對應級別：Grade 5**
**本週預計學習時間：每天約 20 分鐘 × 5 天**

---

## 🎯 本週學習目標

1. 認識次中音譜號（tenor clef）的外形，並能讀出各線、間的音名
2. 掌握中央 C 在次中音譜號上的位置（第四線）
3. 認識六個升降號的大調與關係小調
4. 了解小調第七音的特殊升高方式（還原記號 vs. 重升號）
5. 完成調號在次中音譜號上的書寫位置

---

## 📖 概念講解

### 1. 次中音譜號是什麼？

次中音譜號（tenor clef）與中音譜號（alto clef）外觀完全相同——都是一個形似大寫「B」的記號，稱為「C 譜號」（C clef）。

**關鍵差異：中央 C 的位置不同。**

| 譜號 | 中央 C 的位置 |
|------|--------------|
| 中音譜號（Alto clef）| 第三線 |
| 次中音譜號（Tenor clef）| **第四線** |

這個差異非常重要——只要記住「第四線 = 中央 C」，就能從這個錨點推算所有音名。

[NOTATION:w27_four_clefs]

### 2. 次中音譜號的音名對照

以中央 C（第四線）為基準，往上往下推算：

```
第五線上方一間  = G（高）
第五線          = F（高）
第四間          = E（高）
第四線          = D（高）
第三間          = C（高）—— 即高八度的 C
第三線          = B
第二間          = A
第二線          = G
第一間          = F
第一線          = E
第一線下方一間  = D
第一線下方兩間  = C = 中央 C（第四線位置）
```

等等——讓我重新用正確方式列出（從低到高）：

```
第一線下方二間  = A
第一線下方一間  = B
第一線          = C（中央 C 下方的 C，比中央 C 低一個八度）
第一間          = D
第二線          = E
第二間          = F
第三線          = G
第三間          = A
第四線          = B（中央 C 下方的 B）——等一下
```

依照來源（Sharon Bill）的說明：**中央 C 在第四線**，因此從下往上排列如下：

```
線/間           音名
第四線          = 中央 C (Middle C)
第四間          = D
第五線          = E
第五線上方      = F, G, A, B...
第三間          = B（中央 C 下方的 B）
第三線          = A
第二間          = G
第二線          = F
第一間          = E
第一線          = D
第一線下方      = C（低八度的 C）
```

**記憶訣竅：** 拿到一個次中音譜號的音，先找到第四線（= 中央 C），再用「隔一個位置 = 隔一個音名」往上或往下數。

[NOTATION:w27_tenor_clef]

### 3. 次中音譜號用於哪些樂器？

- **大提琴（Cello）**：演奏高音區時使用
- **長號（Trombone）**：高音區使用
- **低音管（Bassoon）**：偶爾使用

（中音提琴 Viola 使用中音譜號，不要搞混。）

### 4. 調號在次中音譜號上的位置

調號的升降記號，在次中音譜號上的位置會隨之移動（因為音名位置整體改變了）。考試時需要能在次中音譜號的五線譜上，正確書寫或辨認六個升號與六個降號的調號。

### 5. 六個升降號的大小調——Grade 5 完整對照表

| 升降記號數 | 大調 | 關係小調 |
|-----------|------|---------|
| 0（無） | C 大調 | A 小調 |
| 1 升（F#） | G 大調 | E 小調 |
| 2 升（F#, C#） | D 大調 | B 小調 |
| 3 升（F#, C#, G#） | A 大調 | F# 小調 |
| 4 升（F#, C#, G#, D#） | E 大調 | C# 小調 |
| 5 升（F#, C#, G#, D#, A#） | B 大調 | G# 小調 |
| **6 升（F#, C#, G#, D#, A#, E#）** | **F# 大調** | **D# 小調** |
| 1 降（Bb） | F 大調 | D 小調 |
| 2 降（Bb, Eb） | Bb 大調 | G 小調 |
| 3 降（Bb, Eb, Ab） | Eb 大調 | C 小調 |
| 4 降（Bb, Eb, Ab, Db） | Ab 大調 | F 小調 |
| 5 降（Bb, Eb, Ab, Db, Gb） | Db 大調 | Bb 小調 |
| **6 降（Bb, Eb, Ab, Db, Gb, Cb）** | **Gb 大調** | **Eb 小調** |

Grade 5 新增的部分用粗體標示：6 個升號（F# 大調 / D# 小調）與 6 個降號（Gb 大調 / Eb 小調）。

### 6. 小調第七音的特殊升高規則

和聲小音階（harmonic minor scale）需要將第七音**升高半音**。但「升高」的方式取決於該音在調號中的原始狀態：

**情況一：調號中第七音是「降號」**
- 升高半音 → 用**還原記號（♮）** 來取消降號
- 例：D 小調（1 降：Bb）的第七音是 C，調號沒有影響 C，所以升高是 C#

**情況二：調號中第七音是「升號」**
- 升高半音 → 升號再升半音 → 用**重升號（𝄪, double sharp）**
- 例：G# 小調（5 升，其中含 F#）的第七音是 F#，升高後變成 F𝄪（重升 F）

**記憶口訣：**
- 降號第七音 → 還原記號
- 升號第七音 → 重升號

## ✏️ 本週練習題（5 題）

**題 1：**
次中音譜號中，中央 C 在第幾線？

**題 2：**
F# 大調有幾個升號？其關係小調是什麼？

**題 3：**
Gb 大調有幾個降號？其關係小調是什麼？

**題 4：**
G# 小調的和聲小音階第七音是什麼？升高後應寫成什麼記號？

**題 5：**
在次中音譜號上，從第四線（中央 C）往上數三個位置（第四間、第五線、第五間）分別是什麼音名？

---

## ✅ 解答

**題 1：** 第四線（這是次中音譜號與中音譜號最關鍵的差異）。

**題 2：** F# 大調有 **6 個升號**（F#, C#, G#, D#, A#, E#）。關係小調是 **D# 小調**。

**題 3：** Gb 大調有 **6 個降號**（Bb, Eb, Ab, Db, Gb, Cb）。關係小調是 **Eb 小調**。

**題 4：** G# 小調調號含 5 升，其中包含 F#。第七音是 F#（已是升號），升高後變成 **F𝄪（重升 F）**。

**題 5：** 第四間 = D，第五線 = E，第五間 = F。

---

## 💡 考試技巧

1. **看到次中音譜號，先找第四線標上「C」**，再往上/下數音名，不要憑感覺猜。
2. **6 升 = F# 大調**（唸「F sharp 大調」），最後一個升號是 E#——注意不是 E，是 E sharp。
3. **6 降 = Gb 大調**（唸「G flat 大調」），最後一個降號是 Cb——不是 C，是 C flat。
4. 考試中如果看到次中音譜號的音階題，先確認調號正確，再逐音核對，不要一次看完。
5. 小調第七音出題率高——務必記住「降號→還原、升號→重升」這個規則。
