## 📖 概念講解

### 低音譜號（Bass Clef / F Clef）

上週我們學了代表高音域的高音譜號。這週來認識另一個主角——**低音譜號（Bass Clef）**，它又叫做 **F 譜號（F Clef）**。

低音譜號代表**音域較低的聲部**，例如大提琴、低音提琴、鋼琴左手都使用低音譜號。

為什麼叫 F 譜號？因為低音譜號的符號上方有兩個小圓點，這兩個圓點夾住的那條線正是**第四線**，而第四線的音名就是 **F**。這個 F 就是整個低音譜號定位的錨點。

---

### 低音譜號五條線的音名（Lines）

從第一線（最下面）到第五線（最上面）：

```
第五線 → A
第四線 → F （兩個圓點夾住這裡）
第三線 → D
第二線 → B
第一線 → G
```

**英文記憶口訣：Good Boys Deserve Food Always**
- **G**ood → 第一線 G
- **B**oys → 第二線 B
- **D**eserve → 第三線 D
- **F**ood → 第四線 F
- **A**lways → 第五線 A

### 低音譜號四個空間的音名（Spaces）

從最下面的空間到最上面：

```
第四空間（最上） → G
第三空間 → E
第二空間 → C
第一空間（最下） → A
```

**英文記憶口訣：All Cows Eat Grass**
- **A**ll → 第一空間 A
- **C**ows → 第二空間 C
- **E**at → 第三空間 E
- **G**rass → 第四空間 G

「所有的牛都吃草」——非常好記！

### 高音與低音譜號的共同點

兩個譜號的音名排列都遵循同樣的字母順序（A、B、C、D、E、F、G 循環），只是起始位置不同。

特別注意：**中央 C（Middle C）** 在高音譜號中位於第一線下方的加線上，在低音譜號中則位於第五線上方的加線上。同一個音，出現在兩個不同的譜號中！

[NOTATION:w04_middle_c]

### 附點音符（Dotted Notes）

**附點（dot）**是寫在音符右邊的一個小圓點，它的作用是讓音符變長。

**公式：附點音符的時值 = 原時值 + 原時值的一半**

換句話說，加一個點，就等於把原來的音符加上它一半的長度。

| 音符 | 原時值 | 加點後 | 計算 |
|---|---|---|---|
| 附點二分音符（Dotted Minim） | 2 拍 | **3 拍** | 2 + 1 = 3 |
| 附點四分音符（Dotted Crotchet） | 1 拍 | **1½ 拍** | 1 + ½ = 1½ |
| 附點八分音符（Dotted Quaver） | ½ 拍 | **¾ 拍** | ½ + ¼ = ¾ |

**最重要的一個：附點二分音符 = 3 拍**。在 3/4 拍的音樂裡，一個附點二分音符正好可以填滿整個小節！

[NOTATION:w04_dotted]

---

### 連結線（Tie）

**連結線（tie）**是一條連接**兩個相同音高**的音符之間的弧線。它的功能是把兩個音符的時值**加在一起**，使它們聽起來像一個更長的音，中間不中斷。

例如：二分音符（2拍）＋四分音符（1拍），用連結線連起來，就變成一個持續 3 拍的音，等同於附點二分音符的效果。

**重要特徵：**
- 連結線只能連接**音高相同**的兩個音符
- 演奏時第二個音符不重新撥/唱，是原來那個音的延長
- 弧線的兩端分別碰觸兩個音符的音符頭（note head）

---

### 圓滑線（Slur）

**圓滑線（slur）**外觀和連結線很像，也是一條弧線，但**兩者功能完全不同**！

| | 連結線（Tie） | 圓滑線（Slur） |
|---|---|---|
| 連接的音 | 相同音高 | 不同音高（或一連串音符） |
| 功能 | 合併時值，只奏一次 | 要求圓滑奏法（legato） |
| 外觀差異 | 通常只連兩個音 | 可以跨越多個不同高度的音符 |

類比：連結線像「膠帶」把兩塊相同的積木黏在一起，讓它們變成一塊更大的積木；圓滑線像「彩帶」蓋在一排積木上，讓它們整體看起來流暢，但每塊積木仍然獨立。

**判斷訣竅：** 看兩端的音符是不是**同一個音**。相同 = 連結線；不同 = 圓滑線。

## ✏️ 本週練習題

1. 低音譜號的別稱是什麼？它的兩個圓點夾住的是第幾線？那條線的音名是什麼？

2. 根據記憶口訣「All Cows Eat Grass」，低音譜號的第三空間音名是什麼？

3. 計算以下附點音符的時值：
   - (a) 附點二分音符（Dotted Minim）= 幾拍？
   - (b) 附點四分音符（Dotted Crotchet）= 幾拍？

4. 一條弧線連接了兩個都在第三線、同樣是四分音符的音符，請問這是連結線（tie）還是圓滑線（slur）？連起來之後共持續幾拍？

5. 在 3/4 拍的一個小節中，以下哪種組合可以正確填滿這個小節？
   - (A) 一個附點二分音符
   - (B) 兩個二分音符
   - (C) 一個全音符
   - (D) 一個二分音符加一個二分音符

## ✅ 解答
1. 低音譜號的別稱是 **F 譜號（F Clef）**。兩個圓點夾住的是**第四線**，那條線的音名是 **F**。

2. 口訣「All **C**ows **E**at **G**rass」——第三空間對應 E（All=第一空間A，Cows=第二空間C，**Eat=第三空間E**）。所以第三空間的音名是 **E**。

3. (a) 附點二分音符 = 2 + 1 = **3 拍**；(b) 附點四分音符 = 1 + ½ = **1½ 拍**。

4. 兩個音符音高相同（都在第三線），所以這是**連結線（tie）**。兩個四分音符各 1 拍，連起來共持續 **2 拍**。

5. **(A) 正確**。附點二分音符 = 3 拍，正好填滿 3/4 拍的一個小節。
   - (B) 錯：兩個二分音符 = 4 拍，超過 3 拍。
   - (C) 錯：全音符 = 4 拍，超過 3 拍。
   - (D) 錯：同(B)，共 4 拍，超過限制。

## 💡 考試技巧

- **附點計算公式**：「原值加一半」。考試時可以先把原值算出來，再加上一半，不要直接硬背結果。
- **連結線 vs 圓滑線**：看到弧線，第一個動作是看兩端的音高——如果兩個音在**同一個位置**（同一條線或同一個空間），99% 是連結線。
- **低音譜號的 F 定位法**：忘了哪條線是哪個音時，先找到兩個點夾住的第四線（F），然後從 F 向上或向下按字母推算。
- **Grade 1 考試中**，附點二分音符（3拍）是最常出現的附點音符，務必掌握。
