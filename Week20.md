## 📖 概念講解

### 1. 三種譜號的音域定位

你在 Grade 1–3 已學過高音譜號（Treble Clef）和低音譜號（Bass Clef）：

- **高音譜號（Treble Clef）**：代表較高音域的音符
- **低音譜號（Bass Clef）**：代表較低音域的音符
- **中音譜號（Alto Clef）**：代表介於兩者之間的中間音域，用來減少過多加線（ledger lines）的需求

### 2. 中音譜號的外形

中音譜號的符號形狀像一個「B」的變體（有時也稱為 C 譜號，C Clef），由兩段弧線組成。

書寫要點：
- 整個符號寫在五線譜上
- **兩段弧線的匯合處（交叉點）永遠對準第三線**
- 第三線就是中央 C（Middle C）的位置

> 記憶口訣：**中音譜號的「腰」掐在第三線，那裡就是中央 C。**

### 3. 以中央 C 為錨點推算各音

根據原始資料，中音譜號上各線、間的音名如下（由低到高）：

```
第一線下方（加線下）：B
第一線：         C（中央 C 下方的 B → 往上算）
第一間：         D
第二線：         E
第二間：         F
第三線：         G  ← 不對！重新核對：
```

讓我們從中央 C（第三線）往兩側推算：

```
第五線上方（加線）：  D（高音域）
第五線：              C（高）
第五間：              B
第四線：              A
第四間：              G
第三線：              F  ← 不對，讓我依原文重列
```

依原文（Sharon Bill 資料）的音名對照，以中央 C 在第三線為基準：

| 位置 | 音名 |
|------|------|
| 第三線下方第二間 | A |
| 第三線下方第一線（第二線） | B |
| 第三線下方第一間（第二間） | （第二間） |
| **第三線** | **中央 C（Middle C）** |
| 第三線上方第一間（第三間） | D |
| 第四線 | E |
| 第四間 | F |
| 第五線 | G |

完整音名序列（從低到高，對應原文）：
```
...B  C  D  E  F  G  A  B  C  D  E  F  G  A  B  C...
```
其中 **第三線 = 中央 C**，以此為錨點向上向下各數即可。

**實用方法：**
1. 先找到第三線（中央 C）
2. 向上數：D（三間）→ E（四線）→ F（四間）→ G（五線）
3. 向下數：B（二線）→ A（二間，但要注意算法）→ G（一線）

### 4. 調號在中音譜號上的位置

當音樂轉換為中音譜號時，調號中的升降號位置也會相應移動，以對應中音譜號的線間位置。這和高音譜號或低音譜號上的升降號位置不同，但代表的音名完全相同。

### 5. 三種譜號間互換音符的方法

**步驟：**
1. 確認原來譜號中該音的**音名**（例如：高音譜號第三間 = C）
2. 在目標譜號中找到**相同音名**的位置
3. 將音符畫在新位置

**例子：中央 C 在三種譜號的位置：**
- 高音譜號（Treble Clef）：第一加線（first ledger line below）
- 中音譜號（Alto Clef）：**第三線**（middle line）
- 低音譜號（Bass Clef）：第一加線上方（first ledger line above）

## ✏️ 本週練習題

**題 1：** 請說明中音譜號（Alto Clef）的「交叉點」對應到五線譜的哪一條線？那條線代表哪個音？

**題 2：** 在中音譜號中，以中央 C（第三線）為基準，第五線是什麼音？第一線是什麼音？

**題 3：** 以下三種說法，哪一個是正確的？
- (A) 中音譜號主要用於小提琴（violin）
- (B) 中音譜號主要用於中提琴（viola）
- (C) 中音譜號主要用於大提琴（cello）

**題 4：** 如果一個音在高音譜號中位於「第三間」（即 C），請問在中音譜號中，這個 C 應該畫在哪裡？

**題 5：** 為什麼要發明中音譜號？它解決了什麼問題？請用自己的話解釋。

---

## ✅ 解答

**題 1：** 交叉點對應第三線；第三線代表中央 C（Middle C）。

**題 2：** 從第三線（中央 C = C4）向上數：第三間=D、第四線=E、第四間=F、**第五線=G**。向下數：第二間=B、第二線=A（依音階倒推）。向下繼續：第一間、**第一線**對應到更低的音，約為 E 或 F 附近（以中央 C 往下數三個音：B→A→G，所以第一線 = G 下方區域）。精確記法：第一線 = G（低音域）。

**題 3：** **(B) 中音譜號主要用於中提琴（viola）**。原文：「It is chiefly associated with the viola.」

**題 4：** 高音譜號第三間的「C」在中音譜號中應畫在**第三線**（因為那就是中央 C 的位置）。不過需確認是哪個八度的 C；若是中央 C，就在第三線。

**題 5：** 中間音域的音符如果用高音譜號書寫，需要許多加線在下方；如果用低音譜號書寫，則需要許多加線在上方。中音譜號讓這些音符落在五線譜的中央位置，避免大量加線，讓樂譜更易讀。

---

## 💡 考試技巧

1. **記牢第三線 = 中央 C**，這是中音譜號的唯一錨點，其他音都從這裡推算。
2. 考試遇到中音譜號的音名辨認時，**先找第三線標上 C**，再向上或向下按音階順序（C-D-E-F-G-A-B）推算。
3. 書寫中音譜號時，確保兩段弧線的交叉點精確對準第三線，否則整個譜號所代表的音都會錯。
4. 三種譜號互換時，**關鍵是音名不變**，只是在不同譜號上的視覺位置改變了。
